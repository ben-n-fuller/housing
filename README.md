# Housing

Monte Carlo simulation comparing rent vs. buy strategies. Estimates the distribution of net worth difference across variable time horizons and parameter assumptions.

## Setup

```
uv venv && uv pip install -e .
```

Then open the notebooks with the kernel pointed at `.venv/bin/python`.

## Notebooks

- `rent_vs_buy.ipynb` — single-scenario analysis: configures one set of parameters, runs 10,000 Monte Carlo trials, and produces distributional plots and a summary of the buy-vs-rent net worth difference across 1–10 year horizons.
- `rent_vs_buy_sensitivity.ipynb` — parameter sensitivity sweep: sweeps over 192 combinations of appreciation rate, mortgage rate, sell-side transaction cost, and home price. Produces standardized OLS coefficients, partial dependence plots, joint heatmaps, and breakeven horizon analysis. Saves per-combo plots and summaries to `results/`.

## Project Structure

### Core simulation

- `housing/params.py` — parameter dataclasses (`DeterministicParams`, `StochasticParams`), rate path sampling, and defaults
- `housing/buy.py` — buy-side state, derived quantities (mortgage, equity, etc.), and monthly step function
- `housing/rent.py` — rent-side state, derived quantities, and monthly step function
- `housing/simulation.py` — single-trial simulation and Monte Carlo runner

### Analysis and visualization

- `housing/plotting.py` — per-scenario plots (violin, fan chart, P(buy wins), net worth comparison) and aggregate sweep plots (OLS coefficients, partial dependence, joint heatmaps, breakeven horizon)
- `housing/sweep.py` — parameter sweep grid construction, sweep runner, and per-combo artifact export

### Output

- `results/` — auto-generated by the sensitivity sweep; one subdirectory per parameter combination (e.g. `app6_rate65_sell3_price800k/`) containing:
  - `summary.txt` — parameter values and delta summary table
  - `delta_violin.png`, `prob_buy_wins.png`, `delta_fan.png`, `net_worth_comparison.png`
- `results/sweep.csv` — aggregated statistics across all 960 scenario-horizon cells
